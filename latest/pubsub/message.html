class google.cloud.pubsub.message.Message(data, message_id, attributes=None)[source]#

Bases: object

Messages can be published to a topic and received by subscribers.

See https://cloud.google.com/pubsub/docs/reference/rest/v1/PubsubMessage

Parameters:
  • data (bytes) – the payload of the message.
  • message_id (str) – An ID assigned to the message by the API.
  • attributes (dict) – (Optional) Extra metadata associated by the publisher with the message.
